# Interview Scheduler
Interview Scheduler is a single page application (SPA), built with React. Data is persisted by the API server using a PostgreSQL database.It uses jest and cypress testing.

## Dependencies
- Axios
- Classnames
- Normalize.css
- React
- React-dom
- React-scripts
- React-test-renderer

## Getting Started

1. Install all dependencies using the `npm install` command.
2. Go to <http://localhost:8000/> in your browser.


## Running Webpack Development Server

```sh
npm start
```

## Running Jest Test Framework

```sh
npm test
```

## Running Storybook Visual Testbed

```sh
npm run storybook
```
